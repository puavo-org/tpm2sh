Converts keys between ASN.1 formats

Usage: tpm2sh convert [OPTIONS] <INPUT>

Arguments:
  <INPUT>  URI of the input object (e.g., 'file:///path/to/key.pem')

Options:
  -d, --device <DEVICE>          TPM device path [default: /dev/tpmrm0]
      --from <FROM>              Input format [default: pem] [possible values: pem, der]
      --log-format <LOG_FORMAT>  Logging format [default: plain] [possible values: plain, pretty]
      --to <TO>                  Output format [default: der] [possible values: pem, der]
  -p, --password <PASSWORD>      Default authorization password for objects and sessions
  -S, --session <URI>            Session object URI (e.g., 'tpm://0x03000000')
  -h, --help                     Print help
