TPM 2.0 shell

Usage: tpm2sh [OPTIONS] [COMMAND]

Commands:
  convert         Converts keys between ASN.1 formats
  print-error     Prints a TPM error code
  algorithms      Lists available algorithms
  create-primary  Creates a primary key
  delete          Deletes a transient or persistent object
  import          Imports an external key
  load            Loads a TPM key
  objects         Lists TPM objects
  pcr-event       Extends a PCR with an event
  pcr-read        Reads PCRs
  policy          Builds authorization policies
  reset-lock      Resets the dictionary attack lockout timer
  save            Saves to non-volatile memory
  seal            Seals a keyedhash object
  start-session   Starts an authorization session
  unseal          Unseals a keyedhash object
  help            Print this message or the help of the given subcommand(s)

Options:
  -d, --device <DEVICE>          TPM device path [default: /dev/tpmrm0]
      --log-format <LOG_FORMAT>  Logging format [default: plain] [possible values: plain, pretty]
  -p, --password <PASSWORD>      Default authorization password for objects and sessions
  -S, --session <URI>            Session object URI (e.g., 'tpm://0x03000000')
  -h, --help                     Print help
  -V, --version                  Print version
